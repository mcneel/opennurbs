◆ IsValid()

bool ON_UnitSystem::IsValid ( ) const

Returns true if m_unit_system is a valid ON::LengthUnitSystem enum value, which may be ON::LengthUnitSystem::None. If m_unit_system is ON::LengthUnitSystem::CustomUnits, then IsValid() returns true if m_custom_unit_scale > 0.0 and false otherwise. The value of m_custom_unit_name is not tested. See Also: IsSet()
