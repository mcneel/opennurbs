21 /*
22 Description:
23  Convert a number into base32 digits.
24 Parameters:
25  x - [in]
26  x_count - [in]
27  x[] is an array of length x_count and represents the value
28  x[0]*2^(8*(x_count-1)) + ... + x[x_count-2]*256 + x[x_count-1].
29  base32_digits - [out]
30  When base32_digits is not a dynamic array, base32_digits[]
31  must a be an array of length at least
32  ((8*x_count)/5) + (((8*x_count)%5)?1:0) or 1,
33  whichever is greater.
34 
35  The base32_digits[] array will be filled in with base32 digit
36  values (0 to 31) so that the value
37  b[0]*32^(b_count-1) + ... + b[b_count-2]*32 + b[b_count-1]
38  is the same as that defined by the x[] array.
39 Returns
40  The number of base 32 digits in the base32_digits[] array.
41  If 0 is returned, the input is not valid.
42 */
