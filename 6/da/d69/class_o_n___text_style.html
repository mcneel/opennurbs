const unsigned int ON_TextStyle::binary_archive_opennurbs_version

On September 16, 2015 the "V5" ON_Font was split into ON_TextStyle (a document object) and ON_Font (a current runtime resource) = ON_VersionNumberConstruct(6, 0, 2015, 9, 16, 0) Versions of opennurbs before binary_archive_opennurbs_version serialized ON_TextStyle information in the V5 ON_Font format.
